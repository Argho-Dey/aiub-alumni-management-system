# Alumni Management System

This project is a database system designed to manage alumni information and maintain connections between alumni and the institution.

## Features
- Alumni information management
- Event management
- Donation tracking
- Job tracking
- Networking groups

## Database Concepts Used
- ER Diagram
- Normalization (1NF, 2NF, 3NF)
- Relational Schema
- DDL & DML Queries
- Views
- Subqueries
- Joins

## Technologies Used
- Oracle SQL
- JDBC
- Java

## Project Structure
- ER Diagram
- Schema Diagram
- Table Creation
- Data Insertion
- SQL Queries
- Database Connection
